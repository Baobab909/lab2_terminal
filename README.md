## Доступные команды
- **ls** - отображение списка файлов в текущем рабочем каталоге
- **cd** - переход в указанный каталог
- **cat** - вывод содержимого файла в консоль
- **cp** - копирование из источника в назначение
- **mv** - перемещение или переименование файлов/каталогов
- **rm** - удаление файлов и каталогов
- **Логирование** - запись всех действий в файл


### Формат ввода
* **ls**                                 # простой список файлов и папок
* **ls -l**                              # подробный список с правами, размерами и датами
* **cd папка**                           # переход в указанную папку
* **cd ..**                              # переход на уровень выше  
* **cd ~**                               # переход в домашнюю директорию
* **cd**                                 # переход в домашнюю директорию (без аргументов)
* **cat файл.txt**                       # вывод содержимого текстового файла в консоль
* **cp исходный_файл новый_файл**        # копирование файла
* **cp -r исходная_папка новая_папка**   # рекурсивное копирование папки
* **mv старый_файл новый_файл**          # перемещение или переименование файла
* **mv старый_путь новый_путь**          # перемещение между директориями
* **rm файл.txt**                       # удаление файла
* **rm -r папка**                        # удаление папки (требует подтверждения)
* **by**                                 # завершение работы терминала
